# Homelab Documentation

Kompletna dokumentacja prywatnego środowiska sieciowego obejmującego dwie lokalizacje, VPS w chmurze (OVH), segmentację sieci, VPN hub-and-spoke oraz szereg usług uruchomionych lokalnie i zdalnie.

# Homelab — dokumentacja

To repozytorium zawiera dokumentację prywatnego homelabu — topologię, plan adresacji, konfiguracje usług oraz checklisty bezpieczeństwa.

**Status:** Wersja początkowa (polski). Repo przeznaczone do publikacji jako portfolio.

**Licencja:** `LICENSE` (MIT).

## Szybki spis treści

- `docs/overview.md` — podsumowanie architektury i zakresu repo
- `docs/locations/` — opisy lokalizacji (Lokalizacja A, B, VPS)
- `docs/network/` — `topology.md`, `ip-plan.md`, `vpn.md`
- `docs/services/` — instrukcje i notatki dla Pi-hole, Home Assistant, Wazuh, Synology
- `docs/security/` — `hardening-checklist.md` i materiały związane z bezpieczeństwem

## Najważniejsze wskazówki

- Nie umieszczaj w repo haseł, kluczy prywatnych ani plików konfiguracyjnych zawierających sekrety.
- Zgłaszanie problemów: Issues lub Security Advisories na GitHub (więcej w `SECURITY.md`).

Jeśli chcesz, żebym dodał pliki CI, szablony zgłoszeń albo przykładowe playbooki Ansible — daj znać.
