# WAF Concepts (Cloudflare)

Cloudflare WAF nie jest w pełni konfigurowany dla wszystkich domen, jednak:

## Wykorzystywane funkcje:

### 1. Managed Rules (domyślne)

- ochrona przed podstawowymi atakami HTTP,
- sygnatury CVE,
- blokowanie exploitów dla Apache/PHP.

### 2. Firewall Rules

- filtrowanie ruchu po kraju,
- filtrowanie botów,
- ograniczenia dostępu do paneli.

### 3. Bot Fight Mode

- redukcja ruchu automatycznego.

### 4. Turnstile

- ochrona formularzy / numeru telefonu zamieszczonego na stronie.
